# API-тесты для DummyJSON

Автоматизированные тесты для публичного REST API DummyJSON (https://dummyjson.com).

## Ответы на вопросы

### 1. Есть ли опыт ручного тестирования?
Да, есть опыт ручного тестирования веб-приложений, мобильных приложений и API. Проводил функциональное и регрессионное тестирование.

### 2. Есть ли опыт нагрузочного тестирования?
Такого опыта не имею, но рад буду его приобрести.

### 3. Есть ли опыт тестирования безопасности?
Так же опыта в этом не имею.

### 4. Какие инструменты использовались в задании?
- **Python 3.13** - язык программирования
- **pytest 8.0.0** - фреймворк для написания и запуска тестов
- **requests 2.31.0** - HTTP-клиент для отправки запросов к API

## Команда для запуска тестов

```bash
pytest tests/test_api.py -v
